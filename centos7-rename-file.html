CentOS 7 - Rename file

Posted on 화 10 10월 2017 in shell

Circumstance

On developing SW module, I had to rename multiple files to handle requirements. Only files I had consisted with old file name conventions. So, I need to rename files in linux system.

Personal condition

I had a few knowledge about linux shell commands...

What access to solution

  1. I thought to write shell script.
  2. Search keyword 'rename multiple filename' at google.
  3. I got this link. https://stackoverflow.com/questions/16541582/finding-multiple-files-recursively-and-renaming-in-linux
  4. I follow the solution of kamituel. https://stackoverflow.com/users/782609/kamituel

How I did

$ find . -type f -name '*targetExpression*' -exec rename targetExpression newExpression '{}' \;
  • .: Current directory and childrens
  • -type f: I will find node type as file.
  • -name expression: Files with specific expression what I want to find.
  • -exec: Execute command to result of find command.
  • rename: Shell command to rename file.
  • '{}': matched result files of find command. It uses arguments for rename command.
  • \;: Means end of the exec expression.
